# NoRecipeBook
A **Fabric** mod that removes the recipe button from the players inventory.

Vanilla Minecraft | With this mod
----------------- | -------------

![](./media/showcase/craftingtable-with-book.png) | ![](./media/showcase/craftingtable-without-book.png)
